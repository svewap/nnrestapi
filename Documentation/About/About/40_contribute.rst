============
Contribute
============

Contributions are essential to keep open-source projects alive.
We are thankful for any improvement and contribution you make to the TYPO3 Rest Api.

Contribution workflow
^^^^^^^^^^^^^^^^^^^^^

Please always first create an issue at `Bitbucket <https://bitbucket.org/99grad-team/nnrestapi/issues>`__ before starting a change.

Get the latest version from git
"""""""""""""""""""""""""""""""

Fork the `git repository <https://bitbucket.org/99grad-team/nnrestapi/src/master>`__ and create a pull request with your change.

.. code-block:: bash

   git clone https://bitbucket.org/99grad-team/nnrestapi/src/master/
