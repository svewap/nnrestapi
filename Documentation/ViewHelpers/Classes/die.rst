=======================================
die
=======================================

Description
---------------------------------------

<nnt3:die />
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Does nothing except terminate the script.

This can be used to abort the script during Fluid's rendering process.
Handy for debugging mail templates, for example.

.. code-block:: php

	{nnt3:die()}

| ``@return death``
.
