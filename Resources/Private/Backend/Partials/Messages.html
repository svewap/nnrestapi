Problems with the login / web-token?

If there are problems with the authorization of a frontend-user, make sure your server is configured to forward the HTTP-Authorization header. In most cases, simply add this to your .htaccess after the RewriteEngine On line:

# Put this directly after the "RewriteEngine On" command RewriteCond %{HTTP:Authorization} ^(.*) RewriteRule .* - [e=HTTP_AUTHORIZATION:%1]
